Watson Image Generator

This is an image generator to display specific images based on the perceived emotion of user input. It uses both the IBM Watson and Flickr APIs to formulate the result. There is also a chart displayed below the image showing the strength of emotions in the user input. This was a team project in JavaScript class that was harder than it looked because of my uncertainties with PHP and APIs at the time.
